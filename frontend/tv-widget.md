# Tv Widget

- We support tv for both a pair and a single coin.

## Single coin

- Url: https://noodles.fi/tv-widget?coin=<COIN_TYPE>&theme=light|dark

## Pair

- Url: https://noodles.fi/tv-widget?coinA=<COIN_TYPE>&coinB=<COIN_TYPE>&theme=light|dark
